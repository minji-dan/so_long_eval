# so_long_eval

## Install
```
$ make
```

## Running  
```
$ ./so_long [MAP_FILE.ber]
$ ./so_long ./map/normal.ber
```

## Play
|keyboard|action|
|---|---|
|W|move up|
|A|move left|
|S|move down|
|D|move right|
|ESC|close the game window|
